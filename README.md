# Minecraft Panel Pro

Inspired by advanced GitHub repository layouts like OpenClaw.

## Features
- Auto Java install
- PaperMC/Purpur support
- Plugin auto-download
- Termux launcher command
- Multi-folder modular structure

## Install
```bash
git clone YOUR_REPO
cd minecraft-panel-pro
bash install.sh
```

## Usage
```bash
mc -s setup
mc -s start
```
